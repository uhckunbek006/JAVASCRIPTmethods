Array Methods

map

Ар бир элементке функция колдонуп жаңы массив түзөт.

filter

Шартка жооп берген элементтерден жаңы массив түзөт.

reduce

Массивди бир мааниге кошуп чыгарат.

forEach

Ар бир элементке функция колдонулат, жаңы массив түзбөйт.

find

Шартка жооп берген биринчи элементти кайтарат.

some

Шартка жооп берген элемент барбы текшерет.

Loops

for

Классикалык for loop менен массивди же сандык диапазонду айлантуу.

for...in

Объекттин key-терин айлантуу.

for...of

Iterable объектти элементтер менен айлантуу.

while

Шартка жараша цикл иштетет.

do...while

Циклдин денеси бир жолу иштейт, андан соң шарт текшерилет.

map

Прост код snippet жана динамикалык demo:

const numbers=[1,2,3]; const doubled=numbers.map(n=>n*2);

filter

const numbers=[1,2,3,4]; const even=numbers.filter(n=>n%2===0);

reduce

const numbers=[1,2,3,4]; const sum=numbers.reduce((acc,n)=>acc+n,0);

for

for(let i=0;i<5;i++){ console.log(i); }
